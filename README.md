# 🎓 NFT Academic Certificate System

A decentralized app to issue and verify academic certificates using NFTs on Ethereum.

## 💻 Technologies
- Solidity
- Web3.js
- MetaMask
- Ganache
- Remix IDE
- Node.js + Express

## 📁 Structure
- `contracts/` → Smart contracts
- `frontend/` → HTML + Web3 UI
- `backend/` → Express.js API

## 🚀 Getting Started
```bash
cd backend
npm install
node server.js
